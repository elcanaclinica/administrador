Instruções de Uso
⚠️ Como o editor funciona
Não altera o design

O editor lê o HTML atual do site no GitHub e troca apenas o trecho de texto/imagem que você editou. CSS, layout e estrutura permanecem intactos.

Como editar textos

Cada campo mostra o "Texto atual no site" (como está agora) e o "Novo texto" (o que você quer). Ao clicar em Publicar, o editor faz: HTML.replace(atual, novo). Se o "atual" não estiver idêntico ao que está no site, ajuste primeiro.

Botão Recarregar do site

Atualiza os campos "atual" com o que está no GitHub neste momento, caso tenha sido alterado manualmente.

🚀 Primeiros Passos
1. Configure o GitHub

Configurações → cole o token e clique em Salvar e Testar.

2. Edite textos

Textos → ajuste o "Novo texto" → Publicar no Site.

3. Troque imagens

Imagens → escolha a aba → Trocar → selecione a foto.

4. Aguarde

O GitHub Pages atualiza em 1–2 minutos.
